# ARDUINO SOLAR RADIATION SENSOR (DAVIS)
Davis solar radiation sensor used on Arduino UNO with external 3.3V voltage reference on AREF pin

Davis sensor pin:
  Yellow: +3.3V power / 
  Black and red: GND / 
  Brown: data output to analog pin A1 or external ADC

ARDUINO pin:
  AREF to 3.3V / 
  A0 to 3.3 V
